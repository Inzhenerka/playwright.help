# Добро пожаловать

Добро пожаловать в сообщество Playwright. Мы рады видеть вас здесь. Здесь собраны основные материалы, которые помогут в освоении Playwright и его внедрении в работу.

## Навигация

Используйте меню слева для выбора интересующего контента.

## GitHub

В случае проблем, создайте задачу для:

- [Сообщения об ошибках](https://github.com/microsoft/playwright/issues/new?assignees=&labels=&template=bug.md&title=%5BBUG%5D)
- [Запросы на добавление функций](https://github.com/microsoft/playwright/issues/new?assignees=&labels=&template=feature_request.md&title=%5BFeature%5D)
- [Сообщения о регрессии](https://github.com/microsoft/playwright/issues/new?assignees=&labels=&template=regression.md&title=%5BREGRESSION%5D%3A+)
- [Сообщения о уязвимостях безопасности](https://github.com/microsoft/playwright/security/policy)

## Участие

Ознакомьтесь с [руководством по участию](https://github.com/microsoft/playwright/blob/main/CONTRIBUTING.md), если вы хотите внести свой вклад в Playwright.
