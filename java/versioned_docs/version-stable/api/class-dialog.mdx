Объекты [Dialog] отправляются страницей через событие [Page.onDialog(handler)](/api/class-page.mdx#page-event-dialog).

Пример использования класса `Dialog`:

```java
import com.microsoft.playwright.*;

public class Example {
  public static void main(String[] args) {
    try (Playwright playwright = Playwright.create()) {
      BrowserType chromium = playwright.chromium();
      Browser browser = chromium.launch();
      Page page = browser.newPage();
      page.onDialog(dialog -> {
        System.out.println(dialog.message());
        dialog.dismiss();
      });
      page.evaluate("alert('1')");
      browser.close();
    }
  }
}
```

:::note
Диалоги отклоняются автоматически, если только не установлен слушатель [Page.onDialog(handler)](/api/class-page.mdx#page-event-dialog). Когда слушатель присутствует, он **должен** либо [Dialog.accept()](/api/class-dialog.mdx#dialog-accept), либо [Dialog.dismiss()](/api/class-dialog.mdx#dialog-dismiss) диалог - в противном случае страница будет [заморожена](https://developer.mozilla.org/en-US/docs/Web/JavaScript/EventLoop#never_blocking) в ожидании диалога, и такие действия, как клик, никогда не завершатся.
:::

## Методы

### accept {#dialog-accept}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до v1.9</font><x-search>dialog.accept</x-search>

Возвращает, когда диалог был принят.

**Использование**

```java
Dialog.accept();
Dialog.accept(promptText);
```

**Аргументы**
- `promptText` [String] *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="dialog-accept-option-prompt-text"/><a href="#dialog-accept-option-prompt-text" class="list-anchor">#</a>
  
  Текст для ввода в запрос. Не вызывает никаких эффектов, если `type` диалога не является запросом. Опционально.

**Возвращает**
- [void]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="dialog-accept-return"/><a href="#dialog-accept-return" class="list-anchor">#</a>

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до v1.9</font><x-search>dialog.defaultValue</x-search>

**Использование**

```java
Dialog.defaultValue();
```

**Возвращает**
- [String]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="dialog-default-value-return"/><a href="#dialog-default-value-return" class="list-anchor">#</a>
