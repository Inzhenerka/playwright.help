## Введение

:::note

Расширения работают только в Chrome / Chromium, запущенном с постоянным контекстом. Используйте пользовательские аргументы браузера на свой страх и риск, так как некоторые из них могут нарушить функциональность Playwright.
:::

Приведенный ниже фрагмент кода получает [фоновую страницу](https://developer.chrome.com/extensions/background_pages) расширения [Manifest v2](https://developer.chrome.com/docs/extensions/mv2/), исходный код которого находится в `./my-extension`.

Обратите внимание на использование канала `chromium`, который позволяет запускать расширения в безголовом режиме. В качестве альтернативы вы можете запустить браузер в обычном режиме.

```js
const { chromium } = require('playwright');

(async () => {
  const pathToExtension = require('path').join(__dirname, 'my-extension');
  const userDataDir = '/tmp/test-user-data-dir';
  const browserContext = await chromium.launchPersistentContext(userDataDir, {
    channel: 'chromium',
    args: [
      `--disable-extensions-except=${pathToExtension}`,
      `--load-extension=${pathToExtension}`
    ]
  });
  let [backgroundPage] = browserContext.backgroundPages();
  if (!backgroundPage)
    backgroundPage = await browserContext.waitForEvent('backgroundpage');

  // Тестируйте фоновую страницу так же, как и любую другую страницу.
  await browserContext.close();
})();
```

## Тестирование

Чтобы расширение было загружено при запуске тестов, вы можете использовать тестовый фикстур для установки контекста. Вы также можете динамически получить идентификатор расширения и использовать его для загрузки и тестирования всплывающей страницы, например.

Обратите внимание на использование канала `chromium`, который позволяет запускать расширения в безголовом режиме. В качестве альтернативы вы можете запустить браузер в обычном режиме.

Сначала добавьте фикстуры, которые загрузят расширение:

```js title="fixtures.ts"
import { test as base, chromium, type BrowserContext } from '@playwright/test';
import path from 'path';

export const test = base.extend<{
  context: BrowserContext;
  extensionId: string;
}>({
  context: async ({ }, use) => {
    const pathToExtension = path.join(__dirname, 'my-extension');
    const context = await chromium.launchPersistentContext('', {
      channel: 'chromium',
      args: [
        `--disable-extensions-except=${pathToExtension}`,
        `--load-extension=${pathToExtension}`,
      ],
    });
    await use(context);
    await context.close();
  },
  extensionId: async ({ context }, use) => {
    /*
    // для manifest v2:
    let [background] = context.backgroundPages()
    if (!background)
      background = await context.waitForEvent('backgroundpage')
    */

    // для manifest v3:
    let [background] = context.serviceWorkers();
    if (!background)
      background = await context.waitForEvent('serviceworker');

    const extensionId = background.url().split('/')[2];
    await use(extensionId);
  },
});
export const expect = test.expect;
```

Затем используйте эти фикстуры в тесте:

```js
import { test, expect } from './fixtures';

test('пример теста', async ({ page }) => {
  await page.goto('https://example.com');
  await expect(page.locator('body')).toHaveText('Изменено my-extension');
});

test('всплывающая страница', async ({ page, extensionId }) => {
  await page.goto(`chrome-extension://${extensionId}/popup.html`);
  await expect(page.locator('body')).toHaveText('всплывающее окно my-extension');
});
```
