Класс Worker представляет собой [WebWorker](https://developer.mozilla.org/en-US/docs/Web/API/Web_Workers_API). Событие `worker` генерируется на объекте страницы для сигнализации о создании рабочего потока. Событие `close` генерируется на объекте worker, когда рабочий поток завершает свою работу.

```js
page.on('worker', worker => {
  console.log('Worker created: ' + worker.url());
  worker.on('close', worker => console.log('Worker destroyed: ' + worker.url()));
});

console.log('Current workers:');
for (const worker of page.workers())
  console.log('  ' + worker.url());
```

## Методы

### evaluate {#worker-evaluate}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до v1.9</font><x-search>worker.evaluate</x-search>

Возвращает значение, возвращаемое [pageFunction](/api/class-worker.mdx#worker-evaluate-option-expression).

Если функция, переданная в [worker.evaluate()](/api/class-worker.mdx#worker-evaluate), возвращает [Promise], то [worker.evaluate()](/api/class-worker.mdx#worker-evaluate) будет ждать разрешения промиса и вернет его значение.

Если функция, переданная в [worker.evaluate()](/api/class-worker.mdx#worker-evaluate), возвращает не-[Serializable] значение, то [worker.evaluate()](/api/class-worker.mdx#worker-evaluate) вернет `undefined`. Playwright также поддерживает передачу некоторых дополнительных значений, которые не сериализуемы с помощью `JSON`: `-0`, `NaN`, `Infinity`, `-Infinity`.

**Использование**

```js
await worker.evaluate(pageFunction);
await worker.evaluate(pageFunction, arg);
```

**Аргументы**
- `pageFunction` [function] | [string]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-option-expression"/><a href="#worker-evaluate-option-expression" class="list-anchor">#</a>
  
  Функция, которая будет оцениваться в контексте рабочего потока.
- `arg` [EvaluationArgument] *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-option-arg"/><a href="#worker-evaluate-option-arg" class="list-anchor">#</a>
  
  Опциональный аргумент, который будет передан в [pageFunction](/api/class-worker.mdx#worker-evaluate-option-expression).

**Возвращает**
- [Promise]&lt;[Serializable]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-return"/><a href="#worker-evaluate-return" class="list-anchor">#</a>

### evaluateHandle {#worker-evaluate-handle}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до v1.9</font><x-search>worker.evaluateHandle</x-search>

Возвращает значение, возвращаемое [pageFunction](/api/class-worker.mdx#worker-evaluate-handle-option-expression), в виде [JSHandle].

Единственное отличие между [worker.evaluate()](/api/class-worker.mdx#worker-evaluate) и [worker.evaluateHandle()](/api/class-worker.mdx#worker-evaluate-handle) заключается в том, что [worker.evaluateHandle()](/api/class-worker.mdx#worker-evaluate-handle) возвращает [JSHandle].

Если функция, переданная в [worker.evaluateHandle()](/api/class-worker.mdx#worker-evaluate-handle), возвращает [Promise], то [worker.evaluateHandle()](/api/class-worker.mdx#worker-evaluate-handle) будет ждать разрешения промиса и вернет его значение.

**Использование**

```js
await worker.evaluateHandle(pageFunction);
await worker.evaluateHandle(pageFunction, arg);
```

**Аргументы**
- `pageFunction` [function] | [string]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-handle-option-expression"/><a href="#worker-evaluate-handle-option-expression" class="list-anchor">#</a>
  
  Функция, которая будет оцениваться в контексте рабочего потока.
- `arg` [EvaluationArgument] *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-handle-option-arg"/><a href="#worker-evaluate-handle-option-arg" class="list-anchor">#</a>
  
  Опциональный аргумент, который будет передан в [pageFunction](/api/class-worker.mdx#worker-evaluate-handle-option-expression).

**Возвращает**
- [Promise]&lt;[JSHandle]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-handle-return"/><a href="#worker-evaluate-handle-return" class="list-anchor">#</a>

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до v1.9</font><x-search>worker.on('close')</x-search>

Генерируется, когда этот выделенный [WebWorker](https://developer.mozilla.org/en-US/docs/Web/API/Web_Workers_API) завершает свою работу.

**Использование**

```js
worker.on('close', data => {});
```
