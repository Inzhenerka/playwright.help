Объекты [Dialog] отправляются страницей через событие [page.on('dialog')](/api/class-page.mdx#page-event-dialog).

Пример использования класса `Dialog`:

```js
const { chromium } = require('playwright');  // Или 'firefox', или 'webkit'.

(async () => {
  const browser = await chromium.launch();
  const page = await browser.newPage();
  page.on('dialog', async dialog => {
    console.log(dialog.message());
    await dialog.dismiss();
  });
  await page.evaluate(() => alert('1'));
  await browser.close();
})();
```

:::note
Диалоги закрываются автоматически, если нет слушателя [page.on('dialog')](/api/class-page.mdx#page-event-dialog). Когда слушатель присутствует, он **должен** либо [dialog.accept()](/api/class-dialog.mdx#dialog-accept), либо [dialog.dismiss()](/api/class-dialog.mdx#dialog-dismiss) диалог - в противном случае страница будет [заморожена](https://developer.mozilla.org/en-US/docs/Web/JavaScript/EventLoop#never_blocking), ожидая диалог, и такие действия, как клик, никогда не завершатся.
:::

## Методы

### accept {#dialog-accept}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до v1.9</font><x-search>dialog.accept</x-search>

Возвращает, когда диалог был принят.

**Использование**

```js
await dialog.accept();
await dialog.accept(promptText);
```

**Аргументы**
- `promptText` [string] *(необязательный)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="dialog-accept-option-prompt-text"/><a href="#dialog-accept-option-prompt-text" class="list-anchor">#</a>
  
  Текст для ввода в prompt. Не оказывает никакого эффекта, если `type` диалога не является prompt. Необязательный.

**Возвращает**
- [Promise]&lt;[void]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="dialog-accept-return"/><a href="#dialog-accept-return" class="list-anchor">#</a>

### defaultValue {#dialog-default-value}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до v1.9</font><x-search>dialog.defaultValue</x-search>

Если диалог является prompt, возвращает значение по умолчанию. В противном случае возвращает пустую строку.

**Использование**

```js
dialog.defaultValue();
```

**Возвращает**
- [string]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="dialog-default-value-return"/><a href="#dialog-default-value-return" class="list-anchor">#</a>
