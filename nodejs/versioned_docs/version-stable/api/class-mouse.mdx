The Mouse class управляет действиями мыши в CSS-пикселях относительно верхнего левого угла видимой области.

Каждый объект `page` имеет собственный экземпляр Mouse, доступный через [page.mouse](/api/class-page.mdx#page-mouse).

```js
// Использование ‘page.mouse’ для обрисовки квадрата 100x100.
await page.mouse.move(0, 0);
await page.mouse.down();
await page.mouse.move(0, 100);
await page.mouse.move(100, 100);
await page.mouse.move(100, 0);
await page.mouse.move(0, 0);
await page.mouse.up();
```

---

## Методы

### click {#mouse-click}

Является сокращением для [mouse.move()](/api/class-mouse.mdx#mouse-move), [mouse.down()](/api/class-mouse.mdx#mouse-down), [mouse.up()](/api/class-mouse.mdx#mouse-up).

**Использование**

```js
await mouse.click(x, y);
await mouse.click(x, y, options);
```

**Аргументы**
- `x` [number]
  
  X координата относительно видимой области главного фрейма в CSS-пикселях.
- `y` [number]
  
  Y координата относительно видимой области главного фрейма в CSS-пикселях.
- `options` [Object] *(опционально)*
  - `button` "left" | "right" | "middle" *(опционально)*
    
    По умолчанию `left`.
  - `clickCount` [number] *(опционально)*
    
    По умолчанию 1. См. [UIEvent.detail].
  - `delay` [number] *(опционально)*
    
    Время ожидания между `mousedown` и `mouseup` в миллисекундах. По умолчанию 0.

**Возвращает**
- [Promise]&lt;[void]&gt;

---

### wheel {#mouse-wheel}

Отправляет событие `wheel`. Этот метод обычно используется для ручной прокрутки страницы. См. [scrolling](../input.mdx#scrolling) для альтернативных способов прокрутки.

:::note

События колеса могут вызвать прокрутку, если они не обработаны, и этот метод не ожидает завершения прокрутки перед возвратом.
:::

**Использование**

```js
await mouse.wheel(deltaX, deltaY);
```

**Аргументы**
- `deltaX` [number]
  
  Пиксели для горизонтальной прокрутки.
- `deltaY` [number]
  
  Пиксели для вертикальной прокрутки.

**Возвращает**
- [Promise]&lt;[void]&gt;
