Класс Mouse работает в CSS-пикселях основного фрейма относительно верхнего левого угла области просмотра.

Каждый объект `page` имеет свой собственный Mouse, доступный через [page.mouse](/api/class-page.mdx#page-mouse).

```js
// Использование ‘page.mouse’ для обрисовки квадрата 100x100.
await page.mouse.move(0, 0);
await page.mouse.down();
await page.mouse.move(0, 100);
await page.mouse.move(100, 100);
await page.mouse.move(100, 0);
await page.mouse.move(0, 0);
await page.mouse.up();
```

## Методы

### click {#mouse-click}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до v1.9</font><x-search>mouse.click</x-search>

Является сокращением для [mouse.move()](/api/class-mouse.mdx#mouse-move), [mouse.down()](/api/class-mouse.mdx#mouse-down), [mouse.up()](/api/class-mouse.mdx#mouse-up).

**Использование**

```js
await mouse.click(x, y);
await mouse.click(x, y, options);
```

**Аргументы**
- `x` [number]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-option-x"/><a href="#mouse-click-option-x" class="list-anchor">#</a>
  
  X координата относительно области просмотра основного фрейма в CSS-пикселях.
- `y` [number]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-option-y"/><a href="#mouse-click-option-y" class="list-anchor">#</a>
  
  Y координата относительно области просмотра основного фрейма в CSS-пикселях.
- `options` [Object] *(опционально)*
  - `button` "left" | "right" | "middle" *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-option-button"/><a href="#mouse-click-option-button" class="list-anchor">#</a>
    
    По умолчанию `left`.
  - `clickCount` [number] *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-option-click-count"/><a href="#mouse-click-option-click-count" class="list-anchor">#</a>
    
    По умолчанию 1. См. [UIEvent.detail].
  - `delay` [number] *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-option-delay"/><a href="#mouse-click-option-delay" class="list-anchor">#</a>
    
    Время ожидания между `mousedown` и `mouseup` в миллисекундах. По умолчанию 0.

**Возвращает**
- [Promise]&lt;[void]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-return"/><a href="#mouse-click-return" class="list-anchor">#</a>

### wheel {#mouse-wheel}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено в: v1.15</font><x-search>mouse.wheel</x-search>

Отправляет событие `wheel`. Этот метод обычно используется для ручной прокрутки страницы. См. [scrolling](../input.mdx#scrolling) для альтернативных способов прокрутки.

:::note

События колеса могут вызвать прокрутку, если они не обработаны, и этот метод не ждет завершения прокрутки перед возвратом.
:::

**Использование**

```js
await mouse.wheel(deltaX, deltaY);
```

**Аргументы**
- `deltaX` [number]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-wheel-option-delta-x"/><a href="#mouse-wheel-option-delta-x" class="list-anchor">#</a>
  
  Пиксели для горизонтальной прокрутки.
- `deltaY` [number]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-wheel-option-delta-y"/><a href="#mouse-wheel-option-delta-y" class="list-anchor">#</a>
  
  Пиксели для вертикальной прокрутки.

**Возвращает**
- [Promise]&lt;[void]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-wheel-return"/><a href="#mouse-wheel-return" class="list-anchor">#</a>
