JSHandle представляет собой объект JavaScript на странице. JSHandles могут быть созданы с помощью метода [page.evaluate_handle()](/api/class-page.mdx#page-evaluate-handle).

<Tabs
  groupId="python-flavor"
  defaultValue="sync"
  values={[
    {label: 'Sync', value: 'sync'},
    {label: 'Async', value: 'async'}
  ]
}>
<TabItem value="sync">

```py
window_handle = page.evaluate_handle("window")
# ...
```

</TabItem>
<TabItem value="async">

```py
window_handle = await page.evaluate_handle("window")
# ...
```

</TabItem>
</Tabs>

JSHandle предотвращает сборку мусора для ссылочного объекта JavaScript, если только ссылка не будет удалена с помощью [js_handle.dispose()](/api/class-jshandle.mdx#js-handle-dispose). JSHandles автоматически удаляются, когда их исходный фрейм переходит на другую страницу или родительский контекст уничтожается.

Экземпляры JSHandle могут использоваться в качестве аргумента в методах [page.eval_on_selector()](/api/class-page.mdx#page-eval-on-selector), [page.evaluate()](/api/class-page.mdx#page-evaluate) и [page.evaluate_handle()](/api/class-page.mdx#page-evaluate-handle).

### evaluate {#js-handle-evaluate}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до версии v1.9</font><x-search>jsHandle.evaluate</x-search>

Возвращает значение, возвращаемое [expression](/api/class-jshandle.mdx#js-handle-evaluate-option-expression).

Этот метод передает эту ссылку в качестве первого аргумента в [expression](/api/class-jshandle.mdx#js-handle-evaluate-option-expression).

Если [expression](/api/class-jshandle.mdx#js-handle-evaluate-option-expression) возвращает [Promise], то `handle.evaluate` будет ожидать разрешения промиса и вернет его значение.

**Использование**

<Tabs
  groupId="python-flavor"
  defaultValue="sync"
  values={[
    {label: 'Sync', value: 'sync'},
    {label: 'Async', value: 'async'}
  ]
}>
<TabItem value="sync">

```py
tweet_handle = page.query_selector(".tweet .retweets")
assert tweet_handle.evaluate("node => node.innerText") == "10 retweets"
```

</TabItem>
<TabItem value="async">

```py
tweet_handle = await page.query_selector(".tweet .retweets")
assert await tweet_handle.evaluate("node => node.innerText") == "10 retweets"
```

</TabItem>
</Tabs>

**Аргументы**
- `expression` [str]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="js-handle-evaluate-option-expression"/><a href="#js-handle-evaluate-option-expression" class="list-anchor">#</a>
  
  JavaScript выражение для выполнения в контексте браузера. Если выражение оценивается как функция, функция автоматически вызывается.
- `arg` [EvaluationArgument] *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="js-handle-evaluate-option-arg"/><a href="#js-handle-evaluate-option-arg" class="list-anchor">#</a>
  
  Необязательный аргумент для передачи в [expression](/api/class-jshandle.mdx#js-handle-evaluate-option-expression).

**Возвращает**
- [Dict]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="js-handle-evaluate-return"/><a href="#js-handle-evaluate-return" class="list-anchor">#</a>

### evaluate_handle {#js-handle-evaluate-handle}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до версии v1.9</font><x-search>jsHandle.evaluate_handle</x-search>

Возвращает значение, возвращаемое [expression](/api/class-jshandle.mdx#js-handle-evaluate-handle-option-expression) в виде [JSHandle].

Этот метод передает эту ссылку в качестве первого аргумента в [expression](/api/class-jshandle.mdx#js-handle-evaluate-handle-option-expression).

Единственное отличие между `jsHandle.evaluate` и `jsHandle.evaluateHandle` заключается в том, что `jsHandle.evaluateHandle` возвращает [JSHandle].

Если функция, переданная в `jsHandle.evaluateHandle`, возвращает [Promise], то `jsHandle.evaluateHandle` будет ожидать разрешения промиса и вернет его значение.

Смотрите [page.evaluate_handle()](/api/class-page.mdx#page-evaluate-handle) для получения более подробной информации.

**Использование**

```python
js_handle.evaluate_handle(expression)
js_handle.evaluate_handle(expression, **kwargs)
```

**Аргументы**
- `expression` [str]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="js-handle-evaluate-handle-option-expression"/><a href="#js-handle-evaluate-handle-option-expression" class="list-anchor">#</a>
  
  JavaScript выражение для выполнения в контексте браузера. Если выражение оценивается как функция, функция автоматически вызывается.
- `arg` [EvaluationArgument] *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="js-handle-evaluate-handle-option-arg"/><a href="#js-handle-evaluate-handle-option-arg" class="list-anchor">#</a>
  
  Необязательный аргумент для передачи в [expression](/api/class-jshandle.mdx#js-handle-evaluate-handle-option-expression).

**Возвращает**
- [JSHandle]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="js-handle-evaluate-handle-return"/><a href="#js-handle-evaluate-handle-return" class="list-anchor">#</a>

### get_properties {#js-handle-get-properties}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до версии v1.9</font><x-search>jsHandle.get_properties</x-search>

Метод возвращает карту с **именами собственных свойств** в качестве ключей и экземплярами JSHandle для значений свойств.

**Использование**

<Tabs
  groupId="python-flavor"
  defaultValue="sync"
  values={[
    {label: 'Sync', value: 'sync'},
    {label: 'Async', value: 'async'}
  ]
}>
<TabItem value="sync">

```py
handle = page.evaluate_handle("({ window, document })")
properties = handle.get_properties()
window_handle = properties.get("window")
document_handle = properties.get("document")
handle.dispose()
```

</TabItem>
<TabItem value="async">

```py
handle = await page.evaluate_handle("({ window, document })")
properties = await handle.get_properties()
window_handle = properties.get("window")
document_handle = properties.get("document")
await handle.dispose()
```

</TabItem>
</Tabs>

**Возвращает**
- [Map]\[[str], [JSHandle]\]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="js-handle-get-properties-return"/><a href="#js-handle-get-properties-return" class="list-anchor">#</a>

### json_value {#js-handle-json-value}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до версии v1.9</font><x-search>jsHandle.json_value</x-search>

Возвращает JSON представление объекта. Если у объекта есть функция `toJSON`, она **не будет вызвана**.

:::note

Метод вернет пустой JSON объект, если ссылочный объект не может быть преобразован в строку. Он вызовет ошибку, если объект имеет циклические ссылки.
:::

**Использование**

```python
js_handle.json_value()
```

**Возвращает**
- [Dict]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="js-handle-json-value-return"/><a href="#js-handle-json-value-return" class="list-anchor">#</a>
