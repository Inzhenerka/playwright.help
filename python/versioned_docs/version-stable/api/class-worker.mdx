Класс Worker представляет [WebWorker](https://developer.mozilla.org/en-US/docs/Web/API/Web_Workers_API). Событие `worker` генерируется на объекте страницы для сигнализации о создании воркера. Событие `close` генерируется на объекте воркера, когда воркер завершает работу.

```py
def handle_worker(worker):
    print("worker created: " + worker.url)
    worker.on("close", lambda: print("worker destroyed: " + worker.url))

page.on('worker', handle_worker)

print("current workers:")
for worker in page.workers:
    print("    " + worker.url)
```

## Методы

### evaluate {#worker-evaluate}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до версии v1.9</font><x-search>worker.evaluate</x-search>

Возвращает значение, возвращаемое [expression](/api/class-worker.mdx#worker-evaluate-option-expression).

Если функция, переданная в [worker.evaluate()](/api/class-worker.mdx#worker-evaluate), возвращает [Promise], то [worker.evaluate()](/api/class-worker.mdx#worker-evaluate) будет ожидать разрешения промиса и вернет его значение.

Если функция, переданная в [worker.evaluate()](/api/class-worker.mdx#worker-evaluate), возвращает значение, не являющееся [Serializable], то [worker.evaluate()](/api/class-worker.mdx#worker-evaluate) возвращает `undefined`. Playwright также поддерживает передачу некоторых дополнительных значений, которые не сериализуются через `JSON`: `-0`, `NaN`, `Infinity`, `-Infinity`.

**Использование**

```python
worker.evaluate(expression)
worker.evaluate(expression, **kwargs)
```

**Аргументы**
- `expression` [str]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-option-expression"/><a href="#worker-evaluate-option-expression" class="list-anchor">#</a>
  
  JavaScript выражение для выполнения в контексте браузера. Если выражение оценивается как функция, функция автоматически вызывается.
- `arg` [EvaluationArgument] *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-option-arg"/><a href="#worker-evaluate-option-arg" class="list-anchor">#</a>
  
  Опциональный аргумент для передачи в [expression](/api/class-worker.mdx#worker-evaluate-option-expression).

**Возвращает**
- [Dict]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-return"/><a href="#worker-evaluate-return" class="list-anchor">#</a>

### evaluate_handle {#worker-evaluate-handle}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до версии v1.9</font><x-search>worker.evaluate_handle</x-search>

Возвращает значение, возвращаемое [expression](/api/class-worker.mdx#worker-evaluate-handle-option-expression) в виде [JSHandle].

Единственное отличие между [worker.evaluate()](/api/class-worker.mdx#worker-evaluate) и [worker.evaluate_handle()](/api/class-worker.mdx#worker-evaluate-handle) заключается в том, что [worker.evaluate_handle()](/api/class-worker.mdx#worker-evaluate-handle) возвращает [JSHandle].

Если функция, переданная в [worker.evaluate_handle()](/api/class-worker.mdx#worker-evaluate-handle), возвращает [Promise], то [worker.evaluate_handle()](/api/class-worker.mdx#worker-evaluate-handle) будет ожидать разрешения промиса и вернет его значение.

**Использование**

```python
worker.evaluate_handle(expression)
worker.evaluate_handle(expression, **kwargs)
```

**Аргументы**
- `expression` [str]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-handle-option-expression"/><a href="#worker-evaluate-handle-option-expression" class="list-anchor">#</a>
  
  JavaScript выражение для выполнения в контексте браузера. Если выражение оценивается как функция, функция автоматически вызывается.
- `arg` [EvaluationArgument] *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-handle-option-arg"/><a href="#worker-evaluate-handle-option-arg" class="list-anchor">#</a>
  
  Опциональный аргумент для передачи в [expression](/api/class-worker.mdx#worker-evaluate-handle-option-expression).

**Возвращает**
- [JSHandle]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="worker-evaluate-handle-return"/><a href="#worker-evaluate-handle-return" class="list-anchor">#</a>
