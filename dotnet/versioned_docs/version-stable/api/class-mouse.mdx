Класс Mouse работает в CSS-пикселях основного фрейма относительно верхнего левого угла области просмотра.

Каждый объект `page` имеет свой собственный Mouse, доступный через [Page.Mouse](/api/class-page.mdx#page-mouse).

```csharp
await Page.Mouse.MoveAsync(0, 0);
await Page.Mouse.DownAsync();
await Page.Mouse.MoveAsync(0, 100);
await Page.Mouse.MoveAsync(100, 100);
await Page.Mouse.MoveAsync(100, 0);
await Page.Mouse.MoveAsync(0, 0);
await Page.Mouse.UpAsync();
```

## Методы

### ClickAsync {#mouse-click}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до версии v1.9</font><x-search>mouse.ClickAsync</x-search>

Является сокращением для [Mouse.MoveAsync()](/api/class-mouse.mdx#mouse-move), [Mouse.DownAsync()](/api/class-mouse.mdx#mouse-down), [Mouse.UpAsync()](/api/class-mouse.mdx#mouse-up).

**Использование**

```csharp
await Mouse.ClickAsync(x, y, options);
```

**Аргументы**
- `x` [float]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-option-x"/><a href="#mouse-click-option-x" class="list-anchor">#</a>
  
  X координата относительно области просмотра основного фрейма в CSS-пикселях.
- `y` [float]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-option-y"/><a href="#mouse-click-option-y" class="list-anchor">#</a>
  
  Y координата относительно области просмотра основного фрейма в CSS-пикселях.
- `options` `MouseClickOptions?` *(опционально)*
  - `Button` `enum MouseButton { Left, Right, Middle }?` *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-option-button"/><a href="#mouse-click-option-button" class="list-anchor">#</a>
    
    По умолчанию `left`.
  - `ClickCount` [int]? *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-option-click-count"/><a href="#mouse-click-option-click-count" class="list-anchor">#</a>
    
    По умолчанию 1. См. [UIEvent.detail].
  - `Delay` [float]? *(опционально)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-option-delay"/><a href="#mouse-click-option-delay" class="list-anchor">#</a>
    
    Время ожидания между `mousedown` и `mouseup` в миллисекундах. По умолчанию 0.

**Возвращает**
- [void]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-click-return"/><a href="#mouse-click-return" class="list-anchor">#</a>

### WheelAsync {#mouse-wheel}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено в версии: v1.15</font><x-search>mouse.WheelAsync</x-search>

Отправляет событие `wheel`. Этот метод обычно используется для ручной прокрутки страницы. См. [scrolling](../input.mdx#scrolling) для альтернативных способов прокрутки.

:::note

События колеса могут вызвать прокрутку, если они не обработаны, и этот метод не ждет завершения прокрутки перед возвратом.
:::

**Использование**

```csharp
await Mouse.WheelAsync(deltaX, deltaY);
```

**Аргументы**
- `deltaX` [float]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-wheel-option-delta-x"/><a href="#mouse-wheel-option-delta-x" class="list-anchor">#</a>
  
  Пиксели для горизонтальной прокрутки.
- `deltaY` [float]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-wheel-option-delta-y"/><a href="#mouse-wheel-option-delta-y" class="list-anchor">#</a>
  
  Пиксели для вертикальной прокрутки.

**Возвращает**
- [void]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="mouse-wheel-return"/><a href="#mouse-wheel-return" class="list-anchor">#</a>
