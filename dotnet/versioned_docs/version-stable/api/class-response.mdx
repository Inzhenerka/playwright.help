### HeaderValueAsync {#response-header-value}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено в: v1.15</font><x-search>response.HeaderValueAsync</x-search>

Возвращает значение заголовка, соответствующего имени. Имя нечувствительно к регистру. Если несколько заголовков имеют одно и то же имя (кроме `set-cookie`), они возвращаются в виде списка, разделенного `, `. Для `set-cookie` используется разделитель `\n`. Если заголовки не найдены, возвращается `null`.

**Использование**

```csharp
await Response.HeaderValueAsync(name);
```

**Аргументы**
- `name` [string]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="response-header-value-option-name"/><a href="#response-header-value-option-name" class="list-anchor">#</a>
  
  Имя заголовка.

**Возвращает**
- [string]?<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="response-header-value-return"/><a href="#response-header-value-return" class="list-anchor">#</a>

### Headers {#response-headers}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено до v1.9</font><x-search>response.Headers</x-search>

Объект с HTTP-заголовками ответа. Имена заголовков приведены к нижнему регистру. Обратите внимание, что этот метод не возвращает заголовки, связанные с безопасностью, включая заголовки, связанные с cookie. Вы можете использовать [Response.AllHeadersAsync()](/api/class-response.mdx#response-all-headers) для полного списка заголовков, включающего информацию о `cookie`.

**Использование**

```csharp
Response.Headers
```

**Возвращает**
- [IDictionary]&lt;[string], [string]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="response-headers-return"/><a href="#response-headers-return" class="list-anchor">#</a>

---

### HeadersArrayAsync {#response-headers-array}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено в: v1.15</font><x-search>response.HeadersArrayAsync</x-search>

Массив со всеми HTTP-заголовками запроса, связанными с этим ответом. В отличие от [Response.AllHeadersAsync()](/api/class-response.mdx#response-all-headers), имена заголовков НЕ приведены к нижнему регистру. Заголовки с несколькими записями, такие как `Set-Cookie`, появляются в массиве несколько раз.

**Использование**

```csharp
await Response.HeadersArrayAsync();
```

**Возвращает**
- [IReadOnlyList]&lt;`HttpHeader`&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="response-headers-array-return"/><a href="#response-headers-array-return" class="list-anchor">#</a>
  - `name` [string]
    
    Имя заголовка.
  - `value` [string]
    
    Значение заголовка.

### SecurityDetailsAsync {#response-security-details}

<font size="2" style={{position: "relative", top: "-20px"}}>Добавлено в: v1.13</font><x-search>response.SecurityDetailsAsync</x-search>

Возвращает информацию о SSL и других аспектах безопасности.

**Использование**

```csharp
await Response.SecurityDetailsAsync();
```

**Возвращает**
- SecurityDetails?<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="response-security-details-return"/><a href="#response-security-details-return" class="list-anchor">#</a>
  - `issuer` [string]? *(опционально)*
    
    Компонент Common Name из поля Issuer сертификата. Это следует использовать только в информационных целях. Опционально.
  - `protocol` [string]? *(опционально)*
    
    Конкретный используемый протокол TLS (например, `TLS 1.3`). Опционально.
  - `subjectName` [string]? *(опционально)*
    
    Компонент Common Name из поля Subject сертификата. Это следует использовать только в информационных целях. Опционально.
  - `validFrom` [float]? *(опционально)*
    
    Unix-время (в секундах), указывающее, когда сертификат становится действительным. Опционально.
  - `validTo` [float]? *(опционально)*
    
    Unix-время (в секундах), указывающее, когда сертификат становится недействительным. Опционально.
